### 静态博客

### 特点
- [11ty][11ty] 静态文件生成器
- [Sass/SCSS][sass] CSS 预处理器
- [Autoprefixer][autoprefixer] 自动给 CSS 属性添加浏览器厂商前缀
- [Nunjucks][nunjucks] 模板引擎
- [prism][prism] 语法高亮
- [PostCSS][postcss]

[prism]: https://prismjs.com/
